title: Recommended World Settings
---
- **Death Penalty → No Drops** for a relaxed first run; tighten it later if you want tension.
- **Supply Drop Interval → 15–60 min** (default is 3 hours) for more blueprints, accessories and stat boosts.
- **Egg Incubation → instant** while you're learning breeding; restore it later if you want the wait.
- Nudge **gather/drop multipliers** up slightly to soften the resource grind without removing it.
- **Item Weight** can be reduced or disabled if hauling loot home isn't your idea of fun.
- All of these can be changed at any time — experiment freely.
