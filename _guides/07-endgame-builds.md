- **Boss-killer attacker:** Legend + Demon God + Ferocious + the Pal's matching element passive (e.g. Divine Dragon on Jetragon). Swap Ferocious for Musclehead on Pals that never touch base work.
- **24/7 base worker:** Remarkable Craftsmanship + Artisan + Serious + Work Slave — or trade Work Slave for Vampiric/Nocturnal so it never sleeps.
- **Speed mount:** Swift + Runner + Nimble + Eternal Engine. Add Skymarcher instead of Nimble on jump-heavy flyers.
- **Raid tank:** Legend + Diamond Body + Idiosyncratic + Burly Body — near-unflinchable with constant regeneration.
- **Ranch producer:** Diet Lover + Mastery of Fasting + Workaholic + Positive Thinker — work speed does nothing at the ranch; upkeep is everything.

*Full per-Pal recommendations live on the Wiki tab.*
